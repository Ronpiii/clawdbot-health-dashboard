# 2026-02-21

## nightly build: daily log browser (00:00 UTC)

### what
- built `arc log` — browse daily memory files like `git log` for your life
- timeline view with section hierarchy, grep search with context, calendar view, stats dashboard
- parses all memory/YYYY-MM-DD.md files, extracts h2/h3 section structure

### features
- default view: last 7 days, shows sections with subsection counts and line/word counts
- `--grep <pattern>`: search across all logs with 1-line context before/after, highlighted matches
- `--calendar`: monthly grid with logged days marked in brackets (monday-start)
- `--stats`: total lines/words/sections, words/day distribution, current/longest streak, day-of-week chart, monthly chart, recurring topic frequency
- `--sections`: shows subsection hierarchy (h3 under h2)
- `--day YYYY-MM-DD`: full day detail view with formatted content
- `--short`: one-liner per day with top 3 section titles
- date filtering: `--all`, `--week`, `--month`, `--today`, `--days N`, `--since YYYY-MM-DD`
- `--json` for machine-readable output

### first run (28 days of logs)
- 2,398 lines, 16,680 words, 140 sections
- avg 596 words/day, busiest day Feb 3 (1,519 words)
- 28-day unbroken logging streak from Jan 24
- top recurring topics: nightly build (7x), notes (4x), lessons (4x)

### commands
- `arc log` — last 7 days timeline
- `arc log --grep "pattern"` — search with context
- `arc log --calendar` — monthly calendar view
- `arc log --stats` — writing/logging statistics
- `arc log --sections` — skim subsection hierarchy
- `arc log --day YYYY-MM-DD` — full day detail
- `arc log --short` — compact one-liner per day
- aliases: `arc log`, `arc journal`, `arc diary`
