# 5 Keys to Design AWWWARD Winning Websites
**Source:** https://www.youtube.com/watch?v=9TUzpQujYHA
**Speaker:** Adrian Samosa (former lead designer at MediaMonks)
**Transcribed:** 2026-03-03

## Summary

5 keys to designing award-winning websites:

### 1. Use Few Ingredients
- Max 1-2 proven typefaces (not trendy — time-tested ones like Helvetica, Gotham, Universe)
- 2-3 solid colors max
- Few hierarchies (2-4 levels, very distinct from each other)
- Clean, consistent assets (one style — don't mix illustration styles, keep imagery cohesive)

### 2. Master Typography
- It's not the typeface that makes a website look good — it's how you USE it
- Big hooking titles (can feel big through contrast with surrounding whitespace, not just raw size)
- Small text to add depth (contrast between huge title + small body = visual depth, like trees in a forest)
- Clear hierarchy with distinct jumps between levels — not subtle differences
- "It's not the shoe that makes the football player great — it's the player"

### 3. Master Composition
- Keep titles short: 1-5 words max
- Use grids — consistent alignments that repeat throughout the site (rhythm + consistency)
- Use the full canvas — push elements to edges, avoid templateish center-column layouts
- Modulate percentages: vary how much image vs type vs color per section
  - Avoid predictable text-image-text-image patterns
  - Example: 100% image → 60% type → 80% image → 40% text + 20% image
  - Shift between full-bleed images, solid color backgrounds, type-only sections

### 4. Master Animation
- Big things that move on load (hero animations to hook users)
- Animate smaller things on scroll (micro-interactions, parallax, reveal effects)
- Three types: load animations, scroll behavior, micro-interactions (hovers, buttons)
- DON'T animate everything — use animation to reinforce hierarchy
- Animation should direct attention to what's important, not distract

### 5. Master Art Direction
- Do what's RIGHT, not what's TRENDY
- Match assets/style to the brand/project:
  - Product site → clean studio photography
  - Fun/whimsical brand → illustrations
  - Adventure brand → documentary-style photography
- Real example: Oscilloscope brand had yellow as main color, designers tried trendy neon/cyan — looked cool but wrong for the retro brand. Helvetica + yellow was the right choice.
- Think about what's appropriate for THIS project, not what other designers are doing
