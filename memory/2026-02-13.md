## nightly build: arc env (00:00 UTC)

### what
- built `arc env` — environment variable audit dashboard
- scans all projects for .env files (.env, .env.local, .env.example, etc.)
- compares .env.example vs actual .env — finds missing vars, extra undocumented vars
- detects placeholder/unfilled values (your-*, changeme, empty)
- identifies shared keys across projects (same key in 2+ projects)
- checks gitignore coverage — flags exposed .env files with secrets
- per-project breakdown with visual bars and status icons
- health score 0-100 with actionable recommendations
- project filter: `arc env anivia` for single-project detail
- NEVER displays actual secret values — only key names and status

### flags
- `--drift` — detailed drift breakdown by type
- `--shared` — show shared vars across projects
- `--security` — security-focused checks only
- `--short` — one-liner summary
- `--json` — machine-readable output

### why
- natural companion to arc shield (security scanner)
- env drift is a silent productivity killer — you add a var to .env.local but forget .env.example, then wonder why CI fails or a teammate can't onboard
- shared keys across projects mean one rotation = hunt through multiple files
- the tool found real issues: 5 undocumented vars and 2 projects with examples but no actual env files
