## nightly build: past self fortune engine (00:00 UTC)

### what
- built `arc fortune` — extracts wisdom from memory files and surfaces it randomly
- scans MEMORY.md + all daily logs for lessons, insights, principles, technical patterns
- pattern matching for explicit markers (lesson:, key insight:, key finding:) plus section-aware extraction under ## lessons headers
- deduplication with fuzzy prefix stripping (handles "pattern: X" vs "X" dupes)
- 6 auto-detected categories: lesson, insight, principle, technical, security, reflection
- plus manual entries via `--add`
- cached in `memory/fortunes.json`, auto-refreshes when source files change
- boxed display with word-wrapping and attribution line

### commands
- `arc fortune` — one random fortune in a box
- `arc fortune --all` — list everything
- `arc fortune --category` — grouped by type
- `arc fortune --stats` — collection analytics with bars
- `arc fortune --search Q` — search fortunes
- `arc fortune --add "..."` — manually add wisdom
- `arc fortune --refresh` — rebuild cache
- `arc fortune --short` — one-liner
- `arc fortune --json` — machine-readable

### why
- the irony: we log lessons but never re-read them
- this tool closes the loop — passive logs become active reminders
- grows automatically as daily logging continues
- pairs with `arc learn` (capture) → `arc fortune` (retrieve)
- the most self-referential tool yet: your past wisdom, served back to you
