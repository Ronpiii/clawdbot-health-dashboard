# Ventok Lead Database

## Structure

| Field | Description |
|-------|-------------|
| company | Company name |
| industry | manufacturing / logistics / retail / other |
| size | micro (<10) / small (10-49) / medium (50-249) |
| location | City/region in Estonia |
| website | Company URL |
| contact_name | Decision maker name |
| contact_role | Title/position |
| contact_email | Email address |
| contact_phone | Phone (optional) |
| pain_points | Known problems (Excel hell, manual processes, etc.) |
| status | new / contacted / replied / meeting / proposal / won / lost |
| notes | Free text notes |
| source | How we found them |
| added_date | When added to pipeline |
| last_contact | Last outreach date |

## Target Criteria

**Ideal Customer Profile:**
- Estonian SME (10-200 employees)
- Manufacturing, logistics, or wholesale
- Still using Excel/paper for operations
- Growing pains (can't scale current processes)
- Has budget for digitalization (€3k-15k projects)

**Industries to target:**
1. Wood/furniture manufacturing (like TMW)
2. Metal fabrication
3. Food production
4. Logistics/warehousing
5. Automotive services
6. Wholesale/distribution

## Lead Sources
- Estonian Business Registry (e-äriregister)
- Industry associations (Estonian Chamber of Commerce)
- LinkedIn Sales Navigator
- Google Maps (local manufacturers)
- Trade fair exhibitor lists
- Competitors' case studies (who do they serve?)
