# Ventok Cold Outreach Sequence

## Target: Estonian Manufacturers (Wood, Metal, Furniture)

---

## Email 1: Initial Contact
**Send:** Day 0
**Subject options:**
- `[Company] tootmise digitaliseerimine`
- `Küsimus [Company] tootmisprotsesside kohta`
- `Excel vs automaatika - [Company]`

### Estonian Version

```
Tere [Eesnimi],

Leidsin [Company] otsides Eesti [tööstusharu] ettevõtteid, kes võiksid kasu saada tootmisprotsesside digitaliseerimisest.

Töötame praegu sarnase ettevõttega, kes vajas:
• Tootmistellimuste jälgimist reaalajas
• Laoseisu automaatset uuendamist
• Klientide ja tellimuste haldust ühes kohas

Tulemus: 3+ tundi päevas kokkuhoidu ja 0 "kuhu see tellimus jäi?" kõnesid.

Kas teil on 15 minutit sel või järgmisel nädalal, et arutada, kas midagi sarnast võiks [Company] jaoks toimida?

Parimate soovidega,
Ronald Vetter
Ventok OÜ
+372 [number]
ventok.ee
```

### English Translation (for reference)

```
Hi [First name],

I found [Company] while looking for Estonian [industry] companies that could benefit from digitalizing their production processes.

We're currently working with a similar company that needed:
• Real-time production order tracking
• Automatic inventory updates
• Customers and orders managed in one place

Result: 3+ hours saved daily and 0 "where did that order go?" calls.

Do you have 15 minutes this or next week to discuss if something similar could work for [Company]?

Best regards,
Ronald Vetter
Ventok OÜ
```

## Email 2: Follow-up with Value
**Send:** Day 4 (if no reply)
**Subject:** `Re: [Original subject]`

### Estonian Version

```
Tere [Eesnimi],

Kirjutan uuesti, kuna tean, et [tööstusharu] ettevõtetes on kiire aeg.

Üks konkreetne näide: Eesti puidutööstuse ettevõte, kellega töötame, kasutas enne meid:
- Exceli tabeleid tootmise jälgimiseks (aegunud info)
- Käsitsi laoseisu lugemist (vead, ajakulu)
- E-kirju tellimuste kinnitamiseks (kaob ära)

Nüüd: üks süsteem, kus näeb reaalajas mis seisus iga tellimus on, ladu uueneb automaatselt, ja klient saab ise oma tellimuse staatust vaadata.

Kas see kõlab nagu probleem, mis teil ka eksisteerib?

Ronald
```

### English Translation

```
Hi [First name],

Writing again as I know [industry] companies are busy.

One specific example: An Estonian wood industry company we work with previously used:
- Excel spreadsheets for production tracking (outdated info)
- Manual inventory counting (errors, time waste)
- Emails for order confirmations (gets lost)

Now: one system where you see real-time status of every order, inventory updates automatically, and customers can check their order status themselves.

Does this sound like a problem you also have?

Ronald
```

## Customization Notes

### By Industry

**Wood/Furniture:**
- Emphasize: kiln tracking, drying schedules, BOM complexity
- Pain point: partial shipments, custom orders

**Metal Fabrication:**
- Emphasize: project-based production, material tracking
- Pain point: quote-to-production handoff, cutting optimization

**Food Production:**
- Emphasize: batch tracking, expiry dates, compliance
- Pain point: traceability requirements, waste reduction

### Personalization Checklist

Before sending, verify:
- [ ] Correct company name
- [ ] Correct contact first name
- [ ] Industry-specific pain point included
- [ ] Any recent news about company referenced (expansion, new facility, etc.)
- [ ] LinkedIn profile checked for mutual connections
